## ✨ Features

- 🌋 **Geological Data Aggregation**: Access and visualize multi-layer open source geological datasets from various global agencies.
- 🌤️ **Meteorological Data**: Synthesize real-time environmental warnings, climate forecasts, and meteorological reports.
- 💻 **Unified Web Interface**: A clean, single-page web interface to explore interactive dashboards, data catalogs, and hazard maps.
- 🔓 **Open Source Data Sources**: Transparently lists data from open global providers with provenance-first metadata, ensuring you know exactly where your data comes from.

## 🚀 Installation & Setup

Follow these simple steps to get the project running locally on your machine.

### Prerequisites

Ensure you have Node.js and npm installed.

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/geoenv.git
cd geoenv
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm start
```

Your application should now be running at `http://localhost:3000` (or the port specified by your configuration).
